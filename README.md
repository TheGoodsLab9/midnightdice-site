# Midnight Dice Website

A simple GitHub Pages website for Midnight Dice.

## Files

- `index.html` — Home page
- `support.html` — Support page
- `privacy.html` — Privacy Policy page
- `styles.css` — Site styling
- `assets/` — App screenshots

## Setup

1. Create a public GitHub repository.
2. Upload the files from this folder into the root of the repo.
3. Go to Settings → Pages.
4. Source: Deploy from a branch.
5. Branch: main.
6. Folder: / root.
7. Save.

## Before App Store submission

Replace `support@yourdomain.com` in `support.html` and `privacy.html` with your real support email.
